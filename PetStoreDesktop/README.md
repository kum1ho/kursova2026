# PetStoreDesktop - Десктопний додаток для управління зоомагазином

## Опис проекту

Це WPF десктопний додаток (.NET 8) для управління товарами зоомагазину. Додаток використовує Entity Framework Core для роботи з SQL Server та реалізує повний CRUD функціонал для товарів з валідацією даних.

## Особливості

### 🖥️ Інтерфейс
- **Сучасний WPF інтерфейс** українською мовою
- **DataGridView** для відображення списку товарів
- **Пошук** за назвою товару в реальному часі
- **Фільтрація** за категорією
- **Форма редагування** з валідацією даних

### 🔧 Функціонал
- **CRUD операції** для товарів (створення, читання, оновлення, видалення)
- **Валідація даних** (ціна не може бути від'ємною, обов'язкові поля)
- **Пошук та фільтрація** товарів
- **М'яке видалення** (деактивація товарів)
- **Статус залишків** (нормальний/низький залишок)

### 🏗️ Архітектура
- **MVVM Pattern** для розділення логіки та UI
- **Entity Framework Core** для роботи з базою даних
- **Repository Pattern** для доступу до даних
- **Dependency Injection** для керування залежностями

## Моделі даних

### Product
- **ProductID** - унікальний ідентифікатор
- **ProductName** - назва товару (обов'язкове поле)
- **Description** - опис товару
- **Price** - ціна (валідація: > 0)
- **QuantityInStock** - кількість на складі (валідація: ≥ 0)
- **Category** - категорія товару
- **Supplier** - постачальник
- **SKU** - артикул (унікальний)
- **IsActive** - чи активний товар

### Category
- **CategoryID** - ідентифікатор категорії
- **CategoryName** - назва категорії
- **Description** - опис категорії

### Supplier
- **SupplierID** - ідентифікатор постачальника
- **CompanyName** - назва компанії
- **ContactPerson** - контактна особа
- **Phone/Email** - контактні дані

## Валідація даних

### Обов'язкові поля
- Назва товару
- Ціна (> 0)
- Кількість (≥ 0)

### Додаткові перевірки
- Унікальність SKU
- Коректність email
- Валідність телефонних номерів

## Інтерфейс користувача

- **Панель пошуку** за назвою
- **Фільтр** за категорією

## Налаштування та запуск

### Передумови
- .NET 8 SDK
- MS SQL Server (LocalDB або повна версія)

### Налаштування бази даних
1. Перевірте рядок підключення в `appsettings.json`
2. Переконайтесь, що SQL Server запущено

### Запуск проекту
```bash
cd PetStoreDesktop
dotnet run
```
